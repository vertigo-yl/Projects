    \section{How to analyze A/B Testing result with
python?}\label{how-to-analyze-ab-testing-result-with-python}

This is a data challenge case from the book: ``A Collection of Data
Science Take-Home Challenges''. The dataset can be purchased on the
website above, this article is only about solutions and python skills.
All the code is on my Github.

\subsection{1. Experiment Overview}\label{experiment-overview}

Company XYZ is a worldwide e-commerce site with localized versions of
the site. A data scientist at XYZ noticed that Spain-based users have a
much higher conversion rate than any other Spanish-speaking country.

They assumed that one reason could be the translation. All Spanish-
speaking countries had the same translation of the site which was
written by a Spaniard. They agreed to try a test where each country
would have it's one translation written by a local.

After they run the test however, they are really surprised cause the
test is negative. I.e., it appears that the non-localized translation
was doing better!

\textbf{Experimented Change：non-localized to localized translation}
